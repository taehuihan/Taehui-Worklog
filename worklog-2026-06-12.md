### 로고 변환 파이프라인

```
Wikimedia SVG 다운로드
  → sharp (Node.js) trim() + resize(400×400) → 고해상도 PNG
  → Figma upload_assets (submitUrl 획득)
  → curl -X POST {submitUrl} --data-binary @logo.png
  → Figma 컴포넌트에 IMAGE fill 적용
```

**포인트**: `trim()`으로 SVG 여백 제거 후 400×400으로 확대 → 30×30 썸네일에서도 로고가 크게 보임

### 심볼 추출 (KDP)

KDP(Keurig Dr Pepper)의 공식 SVG에는 심볼+워드마크가 한 파일에 있었다. SVG path의 x 좌표를 분석해 좌측 심볼 영역만 `viewBox`로 추출해 사용.

- **최종 결과: 531개 초록 / 33개 미완료**

### 기술적 이슈 및 해결

**문제**: 구형 컴포넌트(2359:xxx 범위)의 티커 이름에 ASCII 0x08 백스페이스 문자가 앞에 붙어 있어 정규식 매칭 실패  
**해결**: `.replace(/[^\w\-\.]/g, '').toUpperCase().trim()` 으로 이름 정제 후 매칭

**문제**: Figma Section 내 노드 `5027:36`을 `getNodeById()`로 접근 불가 (null 반환)  
**해결**: `await figma.setCurrentPageAsync(iconPage)` 로 Icon 페이지 전환 후 접근
